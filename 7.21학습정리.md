# Functions

## 함수 기본
함수의 실행 -> 함수의 호출로 표현
함수정의 : def 이후 함수 이름 작성
함수 정의 : 함수 body/ 콜론 다음 코드블록
반환값 : return문(종료 의미). 없으면 None 반환
함수 호출 : 함수이름(인자1,인자2 ..) 형식

## 함수와 반환값
print함수 - 단순출력, 반환값 없음. (함수로 정의할 시 None반환)
반환과 출력은 다른 개념(결과를 반환한다.)

## 매개변수와 인자
- 매개변수 : 함수를 '정의'할 때 선언하는 이름
- 인자    : 함수를 '호출'할 때 사용값

### 다양한 인자 종류
1. 위치인자
호출 시 인자의 위치에 따름(순서), 개수 부족 시 Typeerror
2. 기본 인자 값
함수 정의 시 매개변수 미리지정.
3. 키워드인자
인자 이름 - 값 매칭하여 전달. 순서 무관
키워드인자는 위치인자 뒤에 위치.(순서의 모호성)
위치인자 - 순서의존, 키워드인자 - 순서파괴. 컴퓨터 판단 침해.
4. 임의의 인자 목록(가변인자)
개수 미지정 위치인자 여러개 / 매개변수 앞 '*' 붙여 사용
5. 임의의 키워드 인자 목록
개수미지정 키워드인자 여러개 / 키워드(dict로묶음)
- 작성순서(권장) : 위치 - 기본 - 가변 - 가변키워드

## 재귀함수
자기자신을 재귀적 호출(반복함수) Ex) 팩토리얼 n!
반복으로 인한 시간초과 위험 -> 종료조건 명확화(스택 오버플로우)

## 함수와 scope
코드 내부 local scope, 그 외 global scope.(자유도 <<<)
변수 수명주기 builtin > global > local

### 이름 검색 규칙(LEGB rule)
Local - Enclosed - Global - Built-in scope 순서로 이름 검색
☆ 내장함수는 이름에 사용 제한
영역별 레벨 인지 및 해석 확인

## global 키워드
변수의 스코프를 전 범위로 지정/ 함수내 전역변수 수정시 사용
- global 키워드 선언 전 참조불가
- 매개변수에 global 키워드 사용불가

## 함수스타일 가이드

### 함수이름 작성규칙
직관성, 함수명은 동사+대상, 쓰기보다 읽기에 좋아야함.
### 단일책임원칙
모든 객체는 하나의 목적과 책임.
함수를 여러개로 나누어 정리(function_08 좋은예 나쁜예 참고)

## 패킹

'*' 튜플 '**' 딕셔너리로 연결
반대방향 동작 -> 언패킹
