# 배쉬 명령어
cd - 위치 변경
.. 상위, . 현재위치


깃 명령어(앞에 git 붙이고 작성)
init - 주의사항 : git 로컬 하위에 새로운 깃 로컬 저장소 만들지 말 것(git init 중복 금지)
복구방법 : 폴더 내 숨겨진 git 폴더를 삭제
status - 상태확인

add 중간영역으로 입력

commit - 변화생성
작성자정보 입력
config - 사용자 인식

git log - 커밋 목록 확인

commit -m first commit 

markdown 
코드를 문서화 하기 위한 언어
본문 : ---하이픈 3개로 구분, 띄어쓰기 4개로 단락구분
**굵게**, *기울기*, ~~취소선~~
이미지 대체텍스트 - 이미지 불러오기 실패 시 출력
이미지 주소복사 - 온라인 이미지 출력

marktext - 무료 마크다운 확장 프로그램

README.md - 사용설명서.가이드북

최초연결 repo - origin별칭 사용

자리 옮기면 github 자격증명 제거 (이사 후 신규등록)
