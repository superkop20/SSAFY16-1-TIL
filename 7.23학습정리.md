# Data structure(구조)
목적에 맞게 데이터를 사용, 관리하기 위한 구조

## 자료구조
효율적인 저장, 관리를 위한 구조를 나눈 것

## 메서드 [데이터 타입 객체.메서드()]
'객체'에 속한 "함수" - 각 데이터 타입별 다양한 기능의 메서드 존재

### 공통 시퀀스 메서드
s.index(x) : 항목 인덱스 반환, s.count(x) : 항복 개수 반환
- 탐색 및 검증 메서드 
1. find x의 첫번째 위치반환 없을 때 -1 반환(에러방지)
2. isup, 대문자 확인
3. islow, 소문자 확인
4. isalpha 문자열이 비지않고 letter일때 true반환

### 문자열 조작 메서드(교재 빨간표)
- replace, strip, split, join(무언가 반환하는 메서드들)
replace 대괄호 - 선택인자라는뜻: (old, new[,count]) new 반
strip - 양쪽 '끝' 특정 문자 제거
split - 문자열 잘라 '리스트'로 만듬
join - (split 반대) 리스트를 하나의 문자열로 합침
- 기타
대소문자 변환 계열 - 규칙만 조금씩 다름
capitalize : 맨 첫 글자만 대문자, 나머지는 소문자
title      : 각 단어의 첫 글자를 대문자
upper      : 전부 대문자
lower      : 전부 소문자
swapcase   : 대<->소문자를 서로 뒤집음

### 리스트 값 추가 삭제 메서드
- append, extend, pop 등
append(x) : 리스트 마지막(맨 뒤)에 항목 x 추가
extend(iterable) : 반복가능한 객체 추가.(풀어서 요소들을 이어붙임)
insert(i,x) : i위치 x 삽입. 음수라면 끝 기준상대적 위치에 삽입
remove : 첫번째로 일치하는 항목 삭제, clear : 비우기
.pop() .pop(i) - '위치'로 꺼내면서 삭제(값 반환 - remove와 다름)
비었으면 마지막값 반환

**append - return 정의되지 않은 함수. print 시 None출력**
**pop은 지운 요소를 return하므로 print Or 정보 사용 가능**

### 정렬
reverse : 거꾸로(정렬아님)
sort : 크기순 정렬(오름차순기본) - reverse=T 옵션 부여시 내림차순

## 객체와 참조
- 복사를 이해하기 위한 전제
변수 할당 : 객체에대한 참조생성과정
새로운 객체 / 기존객체 : 메모리 생성, 기존자료 연결
- 가변객체 : 내용을 변경할 수 있는 객체(ex 리스트)
- 불변객체 : 변경 불가(ex 정수)'갈아타는 방식'
메모리 관리방식 : 가변 - 주소유지, 불변 - 새값 시 새객체 참조
성능최적화 , 메모리효율성 에 있어 각각의 장단점 보유.

### 얕은 복사
객체 최상위 요소만 새로운 메모리에 복사(내부 중첩객체는 참조만 복사)
예시) 리스트 슬라이싱, copy 메서드, list()함수로 복사
- 얕은복사의 한계 : 한 겹만 복사한다 다차원 리스트일경우 문제발생
안쪽에 리스트가 mutual할 시 같은 객체 공유하는 문제.

### 깊은 복사
객체의 모든 수준의 요소를 복사(완전복제) 완전한 독립성 보장
- copy 모듈 import, deepcopy()함수 사용
원본과 복사본은 각각 다른 객체로서 작동 A is B = False
별개의 객체로 평가받기 때문에 다른 하나를 변경해도 영향받지 X

## 참고내용

### List Comprehension(리스트생성방법)
문법 : [표현식 for 변수 in 순회대상 if 조건식(선택사항)] 
기존 - 빈리스트를 만들어야함(비효율) 이후 -> 한줄로 정리
사용예시 : 2차원 행렬 생성 
여러겹 해야할때 * 기호가아니라 for , range를 통해 반복하여 리스트 생성

#### 리스트 생성하는 3가지 방법
1. loop - 반복
2. 리스트 컴프리헨션
3. map 함수
성능 비교 
LC - 대부분 우수, map - 특정상황(내장함수) 가장빠름.
for loop - 복잡한 조건문, 예외처리 필요시 필연적 사용

### 메서드 체이닝
여러 메서드를 연속해서 호출하는 방식 점(.) 을 이어 다른 메서드 호출
앞 메서드가 '값을 반환'해야 뒤로 이을 수 있음(핵심조건)
sort, append함수는 None을 반환하므로 체이닝 불가.
해결책 : 내장함수 사용 / 한줄 씩 이어서 작성

### 문자유형 판별 메서드
숫자판별 3종류 decimal, digit, numeric

### 파이썬 문법 규격(BNF 표기법)
프로그래밍 언어의 문법을 표현하기 위한 표기법

- 콜론 뒤에 짧은 한줄이라면 그 줄에 이어써도 코드 정상 작동
- 단, 2줄이상이라면 줄바꿈및 들여쓰기해야 오류 방지 및 작동
