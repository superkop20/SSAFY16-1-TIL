# 파이썬 Basic syntax 1
"마크다운 다시(-), 숫자번호주석 엔터치면 사라짐."

## 인터프리터
Shell / 확장자 .py인 프로그램에 작성된 파이썬 실행

## 표현식
하나의 값으로 평가될 수 있는 모든 코드
- 변수 : 값을 다시사용하기 위해 붙여주는 고유한 이름(재사용성)
이름 연결 - 할당 / = : 할당 연산자(저장)
변수명 규칙 숙지
1. 숫자시작 금지
2. 알파벳, 언더바, 숫자 구성
3. 일부 할당 금지 변수 존재

### 값 타입 주소정보를 묶은 것을 '객체'
할당문 동작 : Variable = expression 변수 = 결과값
- 재할당 점검

## 타입
데이터 타입 분류 -> 5가지
문자, 숫자, 시퀀스, etc
- 문자열은 불변
f-string : 문자열 형식 지정
Ex) f'{} {}'

## 인덱스
시퀀스 각 값 위치 고유번호 * 0부터 시작

## 슬라이싱
- my_sequence[start:stop:step] 시작, 끝, 간격 지정

## 참고
소수 일부 10진수 -> 2진수 치환시 무한소수 발생.
가장 가까운 근사값 저장
2진수 접두사 0b, 8진수 - 0o, 16진수 0x
표현식 VS 문장 - 하나의 값 Or not
맞춤법 : Style guide (peps.python.org)

# Basic Syntax 2

## 리스트와 튜플
배열(Array)와 리스트 의미는 같다.
리스트 : 변경가능 >=< 문자열
단일요소 튜플 = 후행쉼표. Ex) my_tuple = (1,)
튜플 : 데이터의 안전성과 무결성(불변)

## range
함수로 분류 - 결과 : 정수시퀀스 출력
조건 해당 안하면 에러대신 결과출력 X
range 끝 숫자는 그 전 숫자까지만 출력한다는 뜻. Ex range(1,5) = 1,2,3,4 의미

- 딕셔너리에는 인덱스 X 키 - 밸류 구조.(속도에 강점)
- 세트 : 중괄호
None : Nonetype이라는 타입  

## collection
여러 값을 하나로 묶어 관리하는 자료형 통칭
#문자열, 리스트 튜플 etc.

## 불변과 가변
10분 강의내용 다시보기 복습

## 형변환
- 암시적 형변환(자동)
- 명시적 형변환(직접)

## 연산자
- 산술 연산자(보통의 연산기호들) //(나눗셈 몫), %(나머지)
- 복합 연산자(연산+할당), but 사용예시 말고 실제 풀어쓴 의미사용
가독성을 최대한 높이는 방향
- 비교 연산자(값 비교 후 T/F로 반환) != (같지않음)
is 연산자는 두 객체가 완전히 동일함을 상정하는지 확인할때 사용.
== : 동등성, is : 식별성
- 논리 연산자(and, or, not) 곱 합 부정

## 단축평가
논리연산 두번째 피연산자 계산 X
Or = A가 true라면 B와 관계없이 항상 True 출력
And = 모두 T라면 마지막 값 출력, 앞에서 F면 그 값 출력

## 멤버십연산자
포함관계 확인(in/not in)

## 시퀀스형 연산자
'+' 시퀀스 결합, '*' 시퀀스 반복

* 코드가 아니라 주석을 먼저 작성하여 시나리오 쓰기
